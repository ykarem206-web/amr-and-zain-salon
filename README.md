# ✂️ Amr & Zain Barbershop Booking System

A complete, responsive, and interactive web application for managing barbershop appointments. Built with a modern frontend stack and integrated with cloud services for real-time data handling and email notifications.

![Project Status](https://img.shields.io/badge/Status-Completed-success)
![Version](https://img.shields.io/badge/Version-1.0.0-blue)

## 🌐 Live Demo
[https://amrandzainsalon.netlify.app/]

## ✨ Key Features
* **Dynamic Booking System:** Users can select their preferred barber, date, and available time slots dynamically generated.
* **Real-time Database:** Integrated with **Firebase Firestore** to store, fetch, and manage bookings instantly and prevent double-booking.
* **Email Notifications:** Integrated with **EmailJS** to send automated confirmation and cancellation emails directly to the salon's administration.
* **Cancellation Module:** Users can securely look up their active bookings using their phone numbers and cancel them seamlessly.
* **Modern UI/UX:** Fully responsive, sleek, and dark-themed interface built using **Tailwind CSS** with smooth animations and glass-morphism effects.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (ES6+).
* **Styling:** Tailwind CSS.
* **Backend as a Service (BaaS):** Firebase (Firestore).
* **Third-Party Integrations:** EmailJS (for automated emails).
* **Version Control:** Git & GitHub.
